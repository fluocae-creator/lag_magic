# 展場 QR Code 名片抽選網頁（含入口封面）

## 專案介紹
這是一個給獨立遊戲展場使用的手機網頁。觀眾掃描 QR Code 後，先看到入口封面，點擊進入後抽一張虛擬名片卡。  
此專案為趣味互動，不是抽獎、兌換或賭博系統。
